# Simple Calculator (Kivy Python App)

-----------------------------------------------------------------------------------

⋆OVERVIEW :

Simple Calculator is a basic Kivy Python application that performs standard arithmetic operations such as addition, subtraction, multiplication, and division. The app is built using the Kivy framework, which provides a smooth and responsive graphical user interface (GUI). It is lightweight, easy to use.

-----------------------------------------------------------------------------------

⋆TECHNICAL DETAILS :

TextInput is used as the calculator display to show current input and results.

Button widgets are used for digits, operators, and functional keys.

The BoxLayout organizes buttons and display vertically and horizontally.

-----------------------------------------------------------------------------------

⋆FUNCTIONS :

on_button_press() handles input and operator validation.

on_solution() safely evaluates the entered mathematical expression

-----------------------------------------------------------------------------------

⋆REQUIRMENTS :

Python 3.x

Kivy (Install using: pip install kivy)

-----------------------------------------------------------------------------------

⋆HOW TO RUN :

Save the file as :

Save the file as calculator.py

Open your terminal or command prompt

Navigate to the folder containing the file

A calculator window will open — start performing calculations right away!

-----------------------------------------------------------------------------------

⋆USAGE :

Click on number buttons to enter digits.

Use operator buttons (+, -, *, /) for operations.

Press = to get the result.

Press C to clear the display and start over.

-----------------------------------------------------------------------------------

⋆CONCLUSION :

This Kivy Python Calculator demonstrates how Python and Kivy can be used together to create an interactive and visually appealing GUI application. It’s a great starting project for understanding event handling, layouts, and widget interactions in Kivy.
